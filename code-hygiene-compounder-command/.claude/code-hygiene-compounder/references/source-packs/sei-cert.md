# Source Pack: SEI CERT Secure Coding

- Source ID: `sei-cert`
- Activation: language-specific or security
- Official link: https://wiki.sei.cmu.edu/confluence/spaces/seccode/overview

## Distilled Checks

- For C/C++/Java/Android/Perl or security-sensitive language work, check applicable secure coding rules.
- Prioritize undefined behavior, memory safety, integer handling, resource management, concurrency, serialization, input handling, and unsafe APIs.
- Use severity, likelihood, and remediation cost to prioritize findings.
- Prefer language-native safe constructs over fragile manual handling.
- Record when a finding needs specialist review.
