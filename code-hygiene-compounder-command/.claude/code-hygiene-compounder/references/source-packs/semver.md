# Source Pack: Semantic Versioning

- Source ID: `semver`
- Activation: public package or SDK
- Official link: https://semver.org/

## Distilled Checks

- For public packages/APIs, identify whether a change is patch, minor, or major.
- Preserve documented compatibility unless a major-version change is intended.
- Treat bug fixes as patch-level, backwards-compatible features as minor, and breaking API changes as major.
- Update changelog, migration notes, and version metadata when release behavior changes.
