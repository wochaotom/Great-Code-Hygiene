# Source Pack: CISA Secure by Design

- Source ID: `cisa-secure-by-design`
- Activation: security or product design
- Official link: https://www.cisa.gov/resources-tools/resources/secure-by-design

## Distilled Checks

- Prefer secure defaults and designs that reduce user burden.
- Treat security outcomes as producer responsibility, not merely user configuration.
- Make risky behavior explicit and avoid insecure-by-default options.
- Consider transparency, patchability, vulnerability response, and user impact.
- Favor systemic fixes that remove entire classes of failure.
